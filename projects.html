Doodle Day

This game was made for a 24-hour game jam hosted by Wolfjaw studios. I was a part of a 5 person team and was the technical artist for the game. We used Unity as our engine used its Universal Render Pipeline.

  • Created the CRT shader used to give the game a more retro feeling
  • Created a randomized planet generation shader to create multiple different patterns for each planet
  • Made animations for the boot up, screen shake, and the BONK sprite
